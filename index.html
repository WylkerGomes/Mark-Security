Main Areas of Training

Welcome to Dial Ventures Training Centre

SIA DOOR SUPERVISOR
Security Guarding and Door Supervision

The Door Supervisor Training Course is required to obtain your SIA Door Supervisor Licence. This allows you to work in the private security industry as a Door Supervisor or Security Officer. Door Supervisors usually work at licenced premises but can work in a wide variety of roles including static guarding, manned guarding, key holding etc.


SECURITY GUARD
Security Guarding

The Security Guard Course is required if you want to obtain the SIA Security Guarding Licence which allows you to work in the private security industry as a security officer. Security Officers usually work can work in a wide variety of roles including static guarding, manned guarding, key holding etc.
It is important to understand that Security Guards cannot work as Door Supervisors. However, Door Supervisors can work as Security Guards.
CCTV OPERATOR
Video Surveillance

This course is a legal requirement to obtain the SIA CCTV Licence which enables you to work as a CCTV operator. CCTV Operators usually work in a control room which is connected to a wide array of camera footage. Employers typically include security companies, medium-large corporations, local councils.





CLOSE PROTECTION
Close Protection Officer (CPO)

Being a Close Protection Officer can be extremely demanding and requires in-depth knowledge and training. This means the Close Protection training course covers a broad range of skills, from threat assessment, operational planning and reconnaissance, to vehicle techniques, search procedures and close protection teamwork.

FIRST AID
First Aid at Work

This First Aid course is suitable for staff in any sector, though in this case it is primarily targeted for the security businesses. The course will provide the knowledge and ability to ensure a safe working environment when faced with greater risks, and to treat a wider range of conditions.


EVENT SECURITY
Event security & stewards

Stewards and event security guards operate on all occasions, from major music festivals, concerts to cooperative events. They strive to make the event a success, ensuring that safety is not compromised. The security of customers and the establishment security are the main objective of the events security guard and stewards.
